How do I get an API key?
  1. Log in to your LZT / Lolzteam Market account.
  2. Open your account settings and look for an API, token, developer, or personal access token page.
  3. Create or copy your API token.
  4. Paste the token into the API key field above.
  5. Press Test. If the status returns ready, press Save so this browser remembers it.
Keep the token private. Do not post screenshots of it, do not commit it to GitHub, and remove it with Clear when using a shared computer.
